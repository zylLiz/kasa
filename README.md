# Kasa — Refonte Front-End (React + Vite)

Refonte du site Kasa avec **React**, **React Router** et (à venir) **Sass** + animations CSS.  
Projet basé sur des **maquettes Figma** et un jeu de **données mockées** (`src/data/logements.json`).

## Démarrer le projet

```bash
npm install
npm run dev
```
